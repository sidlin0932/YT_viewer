# Roadmap

本文件記錄專案的未來開發計畫與方向。

## 短期目標 (v0.2.0)

- [ ] **GUI 圖形介面**: 開發簡單的視窗介面，方便使用者設定參數與監控狀態。
- [ ] **更強大的反偵測**: 研究並實作更進階的指紋偽裝技術。
- [ ] **多平台支援**: 確保在 Linux 與 macOS 環境下的穩定性。

## 中期目標 (v0.3.0)

- [ ] **帳號登入支援**: 支援 Google 帳號登入，模擬真實使用者觀看紀錄。
- [ ] **互動增強**: 增加對影片的按讚、訂閱等互動功能 (需登入)。
- [ ] **雲端部署優化**: 提供 Docker Image 與一鍵部署至 Render/Railway 的腳本。

## 長期目標 (v1.0.0)

- [ ] **分散式架構**: 支援多台機器協同運作，由中央伺服器統一控管。
- [ ] **API 介面**: 提供 RESTful API，允許外部程式控制瀏覽器行為。
- [ ] **智慧排程**: 根據目標影片的流量分析，自動調整最佳觀看時段。
