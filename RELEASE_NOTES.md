# Release Notes - v0.1.1

我們很高興宣布 Advanced YouTube Viewer v0.1.1 的首次發布！這是一個專為自動化瀏覽 YouTube 影片而設計的強大工具。

## 🌟 主要功能亮點

- **智慧自動化**: 自動開啟瀏覽器並播放指定影片，無需人工介入。
- **隱私保護**: 完整支援各種類型的代理伺服器 (HTTP/SOCKS5)，隱藏真實 IP。
- **擬人化行為**: 內建隨機滑鼠移動與頁面滾動邏輯，大幅降低被 YouTube 偵測為機器人的風險。
- **彈性排程**: 內建排程系統，可設定每日固定時間或每隔一段時間自動執行任務。

## 🚀 快速開始

1.  安裝依賴: `pip install -r requirements.txt`
2.  設定代理: 在 `proxies.txt` 中填入您的代理列表。
3.  執行程式: `python youtube_viewer_advanced.py`

## ⚠️ 已知問題

- 在某些網路環境下，免費代理可能連線不穩定，建議使用付費的高品質代理。
- 若 YouTube 介面改版，可能會導致部分元素定位失敗，我們將會持續更新修復。

感謝您的使用！如有任何問題，歡迎回報。
